# Data-Cleaning-Reporting-Automation-System

## Project Overview
This project focuses on automating the data cleaning and reporting workflow using Python, Excel, and Power BI.  
The system handles raw data issues such as missing values, duplicate records, and inconsistent formats, then generates interactive reports and dashboards.

## Objective
To improve data quality, reduce manual cleaning effort, and create automated visual reports for better decision-making.

## Tools & Technologies Used

- Python (Pandas, NumPy)
- Microsoft Excel
- Power BI
- Data Visualization

## Project Workflow

Raw Data
↓
Data Cleaning Process
↓
Clean Dataset
↓
Data Quality Report
↓
Power BI Interactive Dashboard

## Data Cleaning Performed

✔ Removed duplicate records  
✔ Handled missing values  
✔ Standardized category and region names  
✔ Fixed inconsistent data formats  
✔ Prepared clean data for visualization
